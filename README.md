# BlackRoot OS

**BlackRoot OS** is a premium, high-tech, hacker-themed terminal simulator, professional code playground, and integrated development sandbox environment built entirely in modern Jetpack Compose for Android. 

It provides an elegant, eye-safe console aesthetic combined with powerful sandbox utilities, a custom Extensions Store containing over 25 upgradeable capabilities, and an interactive database of 100 major CLI commands designed for quick reference, learning, and easy copy-paste interaction.

---

## 🎨 Core Features & Visual Themes

- **Cyberpunk Command Console**: An interactive, dynamic shell simulator supporting essential operations (`help`, `whoami`, `pwd`, `clear`, `ls`, `cat`, `history`), background system health gauges, and customized status indicators.
- **Dynamic Accent Configurator**: Easily toggle accents and themes (such as the orange-red **Sunset Amber Glow** or deep cyan **Deep Ocean Blue**) dynamically via the integrated Extension installer!
- **Universal Code Sandbox (IDE)**: Write, inspect, and preview web structures, Python, or node actions directly on an elegant code-editor canvas.
- **Extensions Store**: A curated registry of 25+ major customization modules, developer compilers, utilities, and simulated security scanners.
- **Copy-Paste Command Hub (100 CLI Commands)**: An extensive repository organized neatly in five dedicated categories (System, Setup, Network, Hacking, Cryptography, and Utilities). Click to copy fully-functional command configurations straight into your device clipboard for quick referencing.

## 📲 How to Download & Install the App on Your Phone

Getting BlackRoot OS onto your Android device is simple and takes less than a minute. Follow these steps:

### Step 1: Generate the APK
1. Open the **Settings** menu in the top right corner of the **Google AI Studio Build** browser window.
2. Select **Generate APK** (or **Build APK**).
3. The platform will automatically package the codebase and compile the secure, signed installation binary.

### Step 2: Download to Your Device
- **From Your Computer**: Once the APK generation completes, click the download link. You can transfer the `.apk` file to your phone using a USB cable, email, or a cloud drive (e.g., Google Drive).
- **Directly on Your Phone**: If you are viewing the AI Studio interface on your phone's web browser, simply tap the download link to download the APK file directly into your device's Downloads directory.

### Step 3: Install on Android
1. Open your phone's **File Manager** (or Files app) and locate the downloaded `BlackRoot.apk` file.
2. Tap the file to begin the installation.
3. *If prompted to allow installations from unknown sources:*
   - Tap **Settings** in the dialog box.
   - Toggle **Allow from this source** (or **Install Unknown Apps**) to enable.
   - Tap **Back** and proceed with the installation.
4. Tap **Install**, wait for the progress bar to complete, and then choose **Open**!

## 🛠️ Tech Stack & Architecture

- **Jetpack Compose**: 100% modern Declarative UI.
- **Kotlin & Coroutines**: High-performance, asynchronous background state flows and delay timers.
- **Material Design 3 (M3)**: Fluid container shapes, accessible click target sizes, and responsive typography pairings.
- **Type-Safe Clipboard Integration**: Immediate physical device feedback and native copy integrations.
